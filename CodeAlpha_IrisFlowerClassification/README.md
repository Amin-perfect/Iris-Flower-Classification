# 🌸 Iris Flower Classification using Machine Learning


## 📌 Project Overview
The goal of this project is to classify Iris flower species using Machine Learning techniques based on flower measurements.

The model predicts the flower species:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

using:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## 🤖 Machine Learning Algorithm
